# imgen

Test image generator

## Usage

```
Usage of ./imgen:
  -h int
        Height of the image (default 200)
  -n int
        Number of images (default 5)
  -t string
        Title of the image (default "test")
  -w int
        Width of the image (default 400)
```
### Example

```
$ imgen -w 400 -h 150 -n 2 -t "test"
```
